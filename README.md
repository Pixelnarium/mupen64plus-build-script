# m64-build-script

A simple bash script for building a complete Mupen64Plus+GLideN64+Angrylion RDP Plus+mupen64plus-gui package.

**Dependencies**

  - SDL2
  - libpng
  - freetype 2
  - zlib
  - Qt5
  - libsamplerate
  - libspeex
  - nasm

After a successful run you find the package inside the release folder.
